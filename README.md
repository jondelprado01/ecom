to setup project:

clone repo
composer install
npm install
create db "ecom"
php artisan migrate
php artisan db:seed


to run project:

run "npm run dev" command to serve all assets (resource/js/scripts.js)

go to /register page to create users other than default user (admin)
then go to /login page


default user:
admin@admin
admin12345

default roles:
1 - Admin
2 - Editor
